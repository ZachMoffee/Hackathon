This README outlines the necessary dependencies and installation steps to run the AI program that predicts mental health illnesses based on text input.
Before running the code, ensure you have the following:
Python Version: Python 3.7 or later.
The following Python libraries are needed:
	•	numpy - For numerical operations.
	•	pandas - For data manipulation and CSV handling.
	•	tensorflow - For building and training the neural network.
	•	scikit-learn - For preprocessing and encoding labels.
If you dont have those: pip install numpy pandas tensorflow scikit-learn
